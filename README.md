
### 追书神器的拙劣复刻版
---

[DEMO](https://web.imhcg.cn/app/book)

---
基于 Vue 和 Vue-router ,实现了查书（搜索、分类、排行榜），看书（基本阅读、目录），收藏（收藏后会记录到书架，实时记录阅读进度，下次打开该书会继续阅读，基于 localStorage ，不清除浏览器记录的话不会丢失记录）。
